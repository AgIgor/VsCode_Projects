📝 Configuration YAML

YAML reference:

gpio:
  - pin: 2
    name: LED
    mode: output  # output | input | input_pullup | input_pulldown
    state: low    # low | high (outputs only)
    delay_on_ms: 500  # ms before turning ON
    delay_off_ms: 200 # ms before turning OFF
    counter:
      edge: falling # falling | rising | change
      debounce_ms: 50

rules:
  - name: MyRule
    trigger: change  # change (default) | always
    if:
      pin: 4
      state: low             # OR counter_gte/lte/eq: N
    then:
      pin: 2
      action: high     # high | low | toggle
      reset_counter: true
    else:
      pin: 2
      action: low

⚠ Use high/low instead of on/off (YAML booleans!).
