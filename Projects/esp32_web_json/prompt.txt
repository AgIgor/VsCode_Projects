
# Projeto: ESP32 Automation Runtime (Máquina Virtual + Linguagem Script)

Você será um arquiteto de software embarcado especialista em ESP32, C++, PlatformIO, FreeRTOS, compiladores e máquinas virtuais.

Seu objetivo é desenvolver um firmware profissional para ESP32 que execute scripts de automação sem necessidade de recompilar o firmware.

## Objetivo

Criar um firmware genérico para ESP32 capaz de interpretar uma linguagem própria semelhante à lógica de PLC/CLP, permitindo que toda a programação seja feita por uma interface Web.

O firmware nunca deverá ser alterado após pronto. Apenas os scripts serão modificados.

O sistema deve ser totalmente modular.

# Máquina Virtual

A linguagem não deverá ser interpretada diretamente.

Ela será compilada para bytecode.

A VM deverá executar instruções sequencialmente usando um Program Counter (PC).

Cada script deve possuir:

```
Program Counter

Estado

Stack

Variáveis locais

Temporizadores

Contadores

Flags

Estado WAIT
```

A VM deverá permitir múltiplos scripts executando simultaneamente.

# Linguagem

Criar uma linguagem semelhante a um PLC.

Exemplo:

```
ON RISING PORTA

OUTPUT LUZ ON

WAIT 30000

OUTPUT LUZ OFF
```

Outro exemplo:

```
IF RPM > 3000

OUTPUT FAN ON

ENDIF
```

Outro exemplo:

```
LOOP 5

OUTPUT LED TOGGLE

WAIT 200

ENDLOOP
```

Outro exemplo:

```
FUNCTION Pisca

OUTPUT LED ON

WAIT 200

OUTPUT LED OFF

WAIT 200

ENDFUNCTION
```
